## 一、产品概述

Stream Analyzer 是一套**让摄像头"看懂"画面**的智能平台。它通过 AI 视觉算法实时分析视频流，在发现异常（如人员闯入、烟火、物品遗留等）时自动告警并生成处理工单，帮助运维团队从"人盯屏幕"升级为"AI 值守、人处理事"的高效工作模式。

**一句话总结**：把视频变成实时警报 + 可追踪的任务工单，让每一个异常都不被遗漏。

## 二、产品核心能力

### 2.1 视频接入与管理

支持将市面上主流摄像头（RTSP、RTMP、HTTP-FLV 协议）快速接入平台，统一管理所有视频流。

- **快速接入**：输入摄像头地址，系统自动探测流信息，3 分钟内完成配置上线
- **状态一目了然**：在线（绿色）、离线（灰色）、异常（红色）状态实时展示
- **批量操作**：同时启用/停用/删除多路视频流

### 2.2 AI 智能检测

平台内置多种 AI 视觉算法，可对视频画面进行实时分析。

| 检测类型 | 适用场景 |
| :--- | :--- |
| **人员入侵** | 周界安防、仓库禁区、施工红线区域 |
| **物品遗留** | 机场/车站可疑包裹、生产线异物 |
| **烟火检测** | 厂房、仓库、森林防火 |
| **区域越界** | 电子围栏、危险区域闯入预警 |

**灵活配置**：可自定义检测区域（画框/画多边形），调节灵敏度，避开树木、人行道等干扰源，大幅降低误报。

### 2.3 实时告警中心

AI 检测到异常后，系统在 **500 毫秒内**将告警推送到用户界面。

- **实时弹窗**：新告警以卡片形式弹出，附带现场截图
- **多维度筛选**：按时间、摄像头、算法类型、处理状态快速定位
- **处置闭环**：支持"确认"、"忽略"、"误报"三种标记，误报数据可反馈优化算法

### 2.4 工单分发管理

将告警事件自动转化为可追踪、可流转的**工单**，形成"告警 → 工单 → 处置 → 闭环"的完整链路。

- **自动生成**：按规则（如置信度≥80%、特定算法类型）自动创建工单，编号唯一可追溯
- **智能分配**：系统根据人员忙闲度、值班表或轮询规则，自动将工单派给最合适的工程师
- **手动指派与转派**：支持指定责任人、多人协办，处理人可申请转派
- **处置验收**：处理人上传现场照片、填写处置记录 → 主管审核验收 → 关闭闭环
- **SLA 时效监控**：高优先级工单须 30 分钟内处理，超时自动升级通知上级
- **看板视图**：以看板（Kanban）形式直观展示各阶段工单，支持拖拽变更状态
- **统计报表**：工单完成率、平均处理时长、SLA 达标率等数据一目了然

### 2.5 人员与权限管理

提供企业级的组织架构和权限控制，确保不同角色各司其职。

- **组织架构**：树形部门管理（如：运维部 → 一值班组 → 二值班组），支持批量导入人员
- **预设角色**：
  - **系统管理员**：平台全部权限
  - **运维主管**：推流/检测/工单管理 + 本部门人员管理
  - **值班工程师**：查看 + 工单处理
  - **业务分析师**：数据报表与知识库维护
- **自定义角色**：按功能模块和操作粒度（查看/创建/编辑/删除/审批）自由组合权限
- **数据权限隔离**：可设置为仅看本人数据、本部门数据或全部数据
- **值班排班**：日历视图排班，支持模板快速生成月度排班，自动记录交接班日志
- **在线状态**：实时显示人员在线/忙碌状态，忙碌与待处理工单数自动关联

### 2.6 工作台驾驶舱

登录后的首页，全局态势一目了然：

- **核心指标**：在线设备数、今日告警数、待处理工单数、平均响应时长
- **实时告警流**：最新告警滚动展示，点击跳转详情
- **快捷操作**：一键新增推流、创建检测、查看告警列表

## 四、适用场景

**场景 1：园区周界安防**
> 摄像头检测到人员翻越围墙 → 系统自动告警并生成工单 → 自动分配给当班巡逻人员 → 巡逻人员到场处置并上传照片 → 主管验收关闭 → 处置方案存入知识库

**场景 2：仓库烟火预警**
> 仓库角落检测到烟雾 → 高置信度告警触发 → 工单以最高优先级自动派发给值班工程师 → 30 分钟内未处理自动通知主管 → 工程师确认是施工灰尘，标记为误报并反馈

**场景 3：工厂生产线异物检测**
> 传送带上遗留工具 → AI 识别后弹窗告警 → 班长手动勾选多条关联告警合并创建工单 → 指派维修人员处理 → 处理后填写处置记录 → 验收归档
